# AWS-Functions
Python code to interact with AWS services i.e. extracting data, loading data

1. [extract data function](https://github.com/Mikekoropsak/AWS-Functions/blob/main/extract%20data%20function): This function will extract all rows from a database contained in athena
2. [load files s3](https://github.com/Mikekoropsak/AWS-Functions/blob/main/load%20file%20to%20S3): Python function to load a file to a S3 bucket
3. [Terminal Commands](https://github.com/Mikekoropsak/AWS-Functions/blob/main/Terminal%20Commands): Various terminal commands for compressing files, transferring files to S3, and loading to environment
